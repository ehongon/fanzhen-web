# 凡真 - 炼体修真综合指导平台

> 我本是道，道本是我，凡本是真，真亦是凡。

凡真是一个整合儒释道医武等各派之长的炼体修真综合指导平台，旨在让普通人也能走上修炼之路。

## 核心特色

- **完整修炼体系**：四阶段九层修炼路径（炼形化精 → 炼精化气 → 炼气化神 → 炼神返虚）
- **凡真七卷**：原创修炼体系，从总纲到实修工具箱，内容详实完整
- **典籍文献库**：7大体系86部典籍，含完整原文，支持逐章阅读
- **功法数据库**：24部功法覆盖7大体系，按阶段/维度/场景/人群分类
- **AI智能辅导**：智能问答与修炼指导
- **社区交流**：心得分享、问答互动、修炼日记
- **游戏化体系**：36级等级、52枚徽章、组队修炼、排行榜
- **用户认证**：完整的注册/登录/找回密码体系
- **持久化存储**：SQLite数据库存储所有数据
- **手环数据预留**：为后续设备对接预留数据表

## 技术栈

- **框架**：Next.js 14 (App Router)
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **UI组件**：shadcn/ui
- **动画**：Framer Motion
- **数据库**：sql.js (SQLite)
- **认证**：JWT

## 修炼体系

### 四阶段九层

1. **炼形化精**（1-9层）：调理身体，培育精气
2. **炼精化气**（10-18层）：运转周天，转化能量
3. **炼气化神**（19-27层）：识神退位，元神显现
4. **炼神返虚**（28-36层）：与道合一，无为而为

### 凡真七卷

- 第一卷：总纲 - 体系缘起与核心理论
- 第二卷：炼形化精 - 筑基之法
- 第三卷：炼精化气 - 周天运转
- 第四卷：炼气化神 - 识神退位
- 第五卷：炼神返虚 - 与道合一
- 第六卷：万法归宗 - 儒释道医武艺整合
- 第七卷：实修工具箱 - 记录模板与自检清单

## 功法覆盖

- **道家**：混元立桩功、涌泉归元法、六字诀、太虚凝神法
- **佛家**：安那般那、止观法门、内观禅、禅宗坐禅、慈悲心法
- **儒家**：儒家静坐、浩然之气培育法
- **武学**：太极拳、形意拳、八卦掌、易筋经、八段锦、五禽戏、大舞
- **中医**：六字诀、五禽戏、自我推拿、强壮功
- **瑜伽**：拜日式、克利亚瑜伽、瑜伽睡眠
- **现代**：正念冥想、身体扫描、渐进式肌肉放松、呼吸工作法
